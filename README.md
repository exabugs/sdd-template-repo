- `copilot-instructions.md`：Copilot の開発ルール（憲法）
- `spec/`：SSOT（requirements / design / tasks）※分割管理
- `docs/adr/`：意思決定履歴（ADR）
- `steering/`：運用ガードレール（規約・品質・開発ルール）
